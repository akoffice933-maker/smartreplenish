# Архитектура SmartReplenish

## Обзор

SmartReplenish — это микросервисная система для прогнозирования спроса и оптимизации товарных запасов.

## Компоненты

### API Gateway
Единая точка входа для всех клиентов. Обрабатывает аутентификацию, rate limiting, маршрутизацию.

### Forecasting Service
Сервис прогнозирования на основе ML (LightGBM + CatBoost ансамбль).

**Основные функции:**
- Построение прогнозов спроса
- Обучение моделей
- Backtesting и валидация

### Optimization Service
Сервис оптимизации заказов.

**Основные функции:**
- Расчет оптимального размера заказа
- Учет ограничений (мин. партия, кратность)
- Расчет страхового запаса

### Alerting Service
Сервис уведомлений и алертов.

**Основные функции:**
- Детекция риска дефицита
- Детекция избытка запасов
- Контроль сроков годности

### Feature Store
Хранилище признаков для ML моделей.

**Основные функции:**
- Вычисление лагов и rolling статистик
- Хранение исторических признаков
- Low-latency доступ к признакам

### Data Ingestion
Сервис загрузки данных.

**Основные функции:**
- Загрузка из внешних источников
- Валидация данных
- Отправка в Kafka и ClickHouse

## Хранение данных

- **ClickHouse**: Временные ряды продаж, прогнозы
- **PostgreSQL**: Метаданные, пользователи, конфигурации
- **Kafka**: Поток событий в реальном времени

## Масштабирование

- Горизонтальное масштабирование микросервисов
- Репликация баз данных
- Шардирование ClickHouse по магазинам
